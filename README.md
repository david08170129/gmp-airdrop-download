# GMP AirDrop Download Website

Static download and instruction website for the GMP AirDrop Windows receiver.

## Files

- `index.html` - one-page website
- `assets/` - GMP AirDrop logo and app icon
- `GMP_Airdrop_Setup.exe` - placeholder Windows installer filename

## Publish

This site uses plain HTML/CSS only. No React, Vite, or build tools are required.

Replace `GMP_Airdrop_Setup.exe` with the real Windows receiver installer before release.
