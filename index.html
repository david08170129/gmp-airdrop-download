How to use

From iPhone to Windows in a few clear steps.

  1. Open GMP AirDrop on your Windows PC.
  2. Click Wireless Receive.
  3. Make sure iPhone and PC are on the same Wi-Fi network.
  4. Open GMP AirDrop on iPhone.
  5. Select photos, videos, or files manually.
  6. Send to the detected PC or scan the QR code shown on Windows.

Privacy Policy

Private by design, local by default.

  • No account is required.
  • No cloud relay is used.
  • Files transfer over your local Wi-Fi network.
  • The user manually chooses every file before sending.

FAQ

Common setup questions.

Does the iPhone app require the Windows receiver?

Yes. For iPhone-to-PC transfer, the Windows receiver must be open and running.

Does it work without internet?

Yes, as long as both devices are on the same local Wi-Fi network.

What if auto discovery does not find my PC?

Use the QR code shown in Wireless Receive, and allow Windows Firewall private-network access.

Reviewer instructions

Testing requires the Windows receiver.

  1. Install or place the Windows receiver on a Windows PC.
  2. Open GMP AirDrop and click Wireless Receive.
  3. Connect the iPhone and PC to the same Wi-Fi network.
  4. Open the iPhone app, select files, and send to the detected PC.
  5. If discovery is blocked, scan the QR code shown on Windows.

SmartScreen explanation

Windows may warn about new apps.

Microsoft SmartScreen may show a warning for new or unsigned builds until the app has enough download reputation. Only install GMP AirDrop from the official download location provided by the developer.

Local Wi-Fi explanation

Your files move device-to-device.

GMP AirDrop creates a temporary local receiver on the Windows PC. The iPhone sends selected files to that receiver over the same Wi-Fi network. No account, cloud storage, or internet relay is required.
